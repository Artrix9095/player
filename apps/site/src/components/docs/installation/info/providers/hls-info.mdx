Embed video content into documents via the native video element. This provider enables
streaming video using the HTTP Live Streaming (HLS) protocol.
[HLS isn't widely supported yet](https://caniuse.com/?search=hls), but
we use the popular [hls.js](https://github.com/video-dev/hls.js) library to ensure it works
anywhere [Media Source Extensions](https://caniuse.com/mediasource) (MSE) are supported, which
accounts for ~96.42% of users tracked on caniuse.
